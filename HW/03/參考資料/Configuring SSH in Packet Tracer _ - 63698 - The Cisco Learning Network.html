        Configuring SSH in Packet Tracer ?

        這個問題已經有回答。

        Josh 2013/11/20 下午 7:52

          Hello,

           

          So just traying to configure SSH in Packet Tracer on my Switch but it doesn't seem to want to work. The commands that I have used:

           

          1. ip domain-name <name>

          2. crypto key generate rsa

          3. ip ssh version 2

          4. username <name> secret <secret>

           

          5. line vty 0 15

          6. transport input ssh

           

          Am I missing something?

           

          Thanks,

           

          Mustillo

          正確答案 透過 just plain old Kev 在 2013/11/20 下午 11:38

          I just tested it, works fine. Two things you may be forgetting...change the default name on your switch, and make sure you turn up the vlan that has the ip you are ssh'ing too. So if you put the ip on interface vlan 1 you need to do

          int vlan 1

          no shut

           

          also, make sure you are using "L" in the ssh command, not the numeral 1...so use

          SSH -l {password} {ip target}

           

          everything else you did is correct

           

          ...also the rsa key size needs to be at least 768 for version 2
